\# CPS Presentation Lab



내가 만든 \*\*CPS UI 프레임워크\*\*와 \*\*Dialogue Command 시스템\*\*을 이용해서  

각종 연출, 컷신, UI 흐름을 실험해 보는 개인용 Unity 프로젝트.



\## 목적



\- CPS UI / Dialogue 패키지를 실제 게임 구조에 붙여보고 검증하기

\- 다양한 연출 패턴(대사, 화면 전환, 카메라 연출, HUD 애니메이션 등)을 샘플 씬으로 축적하기

\- 나중에 포트폴리오 / 데모로 보여줄 수 있는 "연출 카탈로그"를 만드는 것



\## 환경



\- Unity 6000.2.72 (URP 템플릿 기반)

\- 언어: C#



\## 대략 구조



\- `Assets/Scenes/`  

&nbsp; - 연출 실험용 씬들 (예: `IntroDemo`, `BattleCutsceneDemo` 등)

\- `Assets/Game/`  

&nbsp; - 게임 로직, 프레젠테이션 레이어

\- `Packages/com.eondeok.cps.ui/`  

&nbsp; - CPS UI Framework 패키지 (추가 예정)

\- `Packages/com.eondeok.cps.dialogue/`  

&nbsp; - CPS Dialogue Command System 패키지 (추가 예정)



> 이 레포는 개인용 실험 프로젝트이며, 구조나 API는 언제든 바뀔 수 있습니다.
